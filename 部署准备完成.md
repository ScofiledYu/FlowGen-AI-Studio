## 🚀 服务器部署步骤

### 1. 传输文件到服务器

使用 SCP、FTP/SFTP 或 U盘等方式，将上述文件传输到服务器。

### 2. 在服务器上安装依赖（如果使用方式一）

```bash
cd /path/to/deploy/directory
npm install --production
```

### 3. 启动服务器

```bash
# 使用默认端口 3001
node server.js

# 或使用 npm start
npm start
```

### 4. 验证部署

- ✅ 本地访问: `http://localhost:3001`
- ✅ 局域网访问: `http://[服务器IP]:3001`

## ⚠️ 重要提醒

1. **防火墙配置**:
   - Windows: 控制面板 → 防火墙 → 入站规则 → 允许端口 3001
   - Linux: `sudo ufw allow 3001/tcp`

2. **Node.js 版本**:
   - 确保服务器上安装 Node.js v18 或更高版本
   - 检查: `node --version`

3. **网络访问**:
   - 确保服务器可以访问外部 API (`https://models.fangte.com`)
   - 如果在内网，需要配置代理或确保网络连通

4. **端口占用**:
   - 如果端口 3001 被占用，可以通过环境变量修改:
     ```bash
     PORT=8080 node server.js
     ```

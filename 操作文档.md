```bash
npm install
npm run dev
```

### 2. 界面概览

## 🎯 基础操作

### 画布操作

#### 平移画布
- **方法1**：点击工具栏的 Hand Tool 按钮，然后拖拽画布
- **方法2**：按住 `Alt` + 鼠标中键，拖拽画布（松开自动恢复）

#### 缩放画布
- **鼠标滚轮**：向上滚动放大，向下滚动缩小
- **工具栏**：使用缩放控制按钮

#### 选择工具
- **Pointer Tool**：选择、移动、连接节点（默认模式）
- **Hand Tool**：平移画布

#### 自动布局
- **快捷键**：按 `L` 键
- **功能**：自动排列所有节点，保持工作流整洁

### 视图控制

- **小地图**：右下角显示画布概览
- **适应视图**：双击小地图或使用工具栏按钮
- **背景网格**：可切换显示/隐藏

## 📦 节点操作

### 创建节点

#### 方法1：从侧边栏拖拽
1. 在左侧边栏选择节点类型
2. 拖拽到画布上
3. 释放鼠标创建节点

#### 方法2：从故事板创建
1. 在左侧故事板区域选择图片
2. 点击图片上的"创建节点"按钮
3. 自动创建输入节点并关联图片

#### 方法3：右键菜单
1. 在画布空白处右键
2. 选择"添加节点"
3. 选择节点类型

### 节点类型说明

#### 1. 输入节点 (INPUT)
- **用途**：故事板图片输入
- **功能**：
  - 显示故事板图片预览
  - 支持多图关联
  - 生成的内容会显示为缩略图

#### 2. 处理节点 (PROCESSOR)
- **用途**：AI分析处理
- **功能**：
  - Qwen多模态图像分析
  - 生成图像描述
  - 提供创作建议

#### 3. 输出节点 (OUTPUT)
- **用途**：图像生成
- **功能**：
  - Nano Banana Pro 图像生成
  - 支持多图参考
  - 自定义宽高比和分辨率

#### 4. 视频节点 (MOV)
- **用途**：视频生成
- **功能**：
  - 可灵视频生成
  - 首尾帧控制
  - 质量、时长、创意度调节

### 节点编辑

#### 选择节点
- **单击**：选择单个节点
- **框选**：拖拽选择多个节点
- **Ctrl/Cmd + 点击**：多选节点

#### 移动节点
- **拖拽**：直接拖拽节点移动位置
- **方向键**：选中节点后使用方向键微调

#### 删除节点
- **Delete/Backspace**：选中节点后按删除键
- **右键菜单**：选择"删除节点"

#### 复制节点
- **右键菜单**：选择"复制节点"
- **快捷键**：Ctrl+C / Cmd+C（未来支持）

### 节点连接

#### 创建连接
1. 将鼠标悬停在节点的连接点上
2. 按住鼠标左键拖拽
3. 拖到目标节点的连接点
4. 释放鼠标完成连接

#### 连接规则
- **输入节点** → **处理节点**：图片分析
- **输入节点** → **输出节点**：图像生成
- **输出节点** → **视频节点**：视频生成
- **处理节点** → **输出节点**：使用分析结果生成

#### 删除连接
- **选中连接线**：点击连接线选中
- **Delete键**：按删除键删除连接

## 🔗 工作流构建

### 基础工作流

#### 工作流1：图像生成
```
输入节点 → 输出节点
```
1. 创建输入节点，添加故事板图片
2. 创建输出节点
3. 连接输入节点到输出节点
4. 配置输出节点参数
5. 运行输出节点生成图像

#### 工作流2：图像分析+生成
```
输入节点 → 处理节点 → 输出节点
```
1. 创建输入节点，添加图片
2. 创建处理节点，连接输入节点
3. 运行处理节点分析图片
4. 创建输出节点，连接处理节点
5. 使用分析结果生成图像

#### 工作流3：图像生成+视频生成
```
输入节点 → 输出节点 → 视频节点
```
1. 创建输入节点，添加图片
2. 创建输出节点，生成图像
3. 创建视频节点，连接输出节点
4. 配置视频参数
5. 运行视频节点生成视频

### 复杂工作流

#### 多图参考生成
```
输入节点1 ┐
输入节点2 ├→ 输出节点
输入节点3 ┘
```
- 多个输入节点连接到同一个输出节点
- 输出节点会使用所有输入节点的图片作为参考

#### 批量处理
```
输入节点 → 输出节点1
        → 输出节点2
        → 输出节点3
```
- 一个输入节点连接多个输出节点
- 可以同时生成多个不同风格的图像

## 🎨 图像生成

### 使用 Nano Banana Pro

#### 1. 创建输出节点
- 从侧边栏拖拽或右键菜单创建

#### 2. 配置参数

**基础设置**：
- **模型选择**：选择 "Nano Banana Pro(生图)"
- **创意描述**：输入想要生成的图像描述
- **负面提示词**：输入不想要的内容

**生成设置**：
- **宽高比**：1:1、16:9、9:16、4:3、3:4
- **分辨率**：标准、高清
- **生成数量**：1张、2张、3张、4张

**参考图片**：
- **主图**：从输入节点获取
- **多图参考**：点击"多图参考"添加更多参考图片
  - 从本地上传
  - 从库文件选择

#### 3. 运行生成
- 点击节点上的"运行"按钮
- 或选中节点后按空格键
- 等待生成完成

#### 4. 查看结果
- 生成完成后，图像会显示在节点上
- 点击节点查看大图预览
- 在详情面板可以下载或复制URL

### 多图参考技巧

1. **风格一致性**：使用相似风格的参考图
2. **构图参考**：使用想要的构图作为参考
3. **细节补充**：用参考图补充细节信息
4. **数量控制**：建议使用2-4张参考图，过多可能影响效果

## 🎬 视频生成

### 使用可灵 (Keling)

#### 1. 创建视频节点
- 从侧边栏拖拽或右键菜单创建

#### 2. 配置参数

**基础设置**：
- **模型选择**：选择 "可灵(生视频)"
- **创意描述**：输入视频描述
- **负面提示词**：输入不想要的内容

**首尾帧设置**：
- **首帧图片**：视频开始画面
  - 从本地上传
  - 从库文件选择
  - 从输入节点获取
- **尾帧图片**：视频结束画面
  - 从本地上传
  - 从库文件选择
  - 从输入节点获取

**生成设置**：
- **质量**：高质量、标准
- **时长**：5秒、10秒
- **创意度**：0-100滑块调节
- **生成数量**：1条、2条、3条、4条

#### 3. 运行生成
- 点击节点上的"运行"按钮
- 等待生成完成（可能需要较长时间）

#### 4. 查看结果
- 生成完成后，视频会显示在节点上
- 点击节点播放视频
- 在详情面板可以下载或复制URL

### 首尾帧技巧

1. **连贯性**：首尾帧应该有一定的连贯性
2. **变化幅度**：创意度控制变化幅度
3. **构图**：首尾帧的构图要合理
4. **风格**：保持风格一致性

## 🤖 AI助手使用

### 打开AI助手
- 点击右侧面板的"AI助手"标签
- 或使用快捷键（如果有）

### 基础对话

#### 1. 文本对话
- 在输入框输入问题
- 点击发送或按Enter键
- AI会基于当前上下文回答

#### 2. 图片分析
- 点击"附加图片"按钮
- 选择要分析的图片
- 输入分析需求
- AI会分析图片内容

#### 3. 多图分析
- 点击"附加图片"按钮
- 选择多张图片
- 输入分析需求
- AI会对比分析多张图片

### 节点引用

#### 引用节点
1. 在画布上选择要引用的节点
2. 点击AI助手中的"引用节点"按钮
3. 选中的节点会显示在引用区域
4. AI会基于引用的节点内容回答

#### 停止引用
- 点击"中止引用"按钮
- 清除所有引用的节点

### 节点预览
- 选中节点后，AI助手会自动显示节点预览
- 预览图会显示在"当前选中节点"区域
- 可以基于预览图进行对话

### 使用技巧

1. **明确需求**：描述要清晰具体
2. **提供上下文**：引用相关节点提供上下文
3. **多轮对话**：可以基于AI的回答继续提问
4. **应用建议**：可以将AI的建议应用到节点配置中

## 💾 工程管理

### 自动保存
- **功能**：自动保存到浏览器本地存储
- **触发**：节点或边发生变化时自动保存
- **恢复**：刷新页面后自动恢复

### 手动保存

#### 保存工程
1. 点击工具栏的"保存"按钮（💾图标）
2. 输入文件名（可选）
3. 选择保存位置
4. 点击保存

#### 加载工程
1. 点击工具栏的"加载"按钮（📁图标）
2. 选择要加载的JSON文件
3. 确认加载（会覆盖当前工程）

#### 清空画布
1. 点击工具栏的"清空"按钮（🗑️图标）
2. 确认清空操作
3. 画布会重置为空状态

### 节点导出

#### 导出选中节点
1. 选择要导出的节点（单选或多选）
2. 右键点击选中的节点
3. 选择"导出节点"
4. 选择保存位置
5. 节点会导出为JSON文件

#### 导出用途
- **备份**：备份重要节点配置
- **分享**：分享节点配置给他人
- **复用**：在其他工程中复用节点

### 撤销/重做

#### 撤销操作
- **快捷键**：`Ctrl+Z` (Windows/Linux) 或 `Cmd+Z` (Mac)
- **功能**：撤销上一步操作
- **限制**：最多支持50步历史记录

#### 重做操作
- **快捷键**：`Ctrl+Y` (Windows/Linux) 或 `Cmd+Shift+Z` (Mac)
- **功能**：重做被撤销的操作

#### 注意事项
- 在输入框中输入时，撤销/重做不会触发（避免干扰文本输入）
- 历史记录在刷新页面后会清空

## 🎓 高级技巧

### 1. 批量生成技巧

- 创建多个输出节点
- 配置不同的参数

#### 节点分组
- 将相关节点放在一起
- 使用自动布局整理
- 保持工作流清晰

#### 连接优化
- 避免交叉连接
- 使用合理的连接顺序
- 定期整理布局

### 3. 参数调优

#### 图像生成参数
- **创意描述**：详细描述效果更好
- **负面提示词**：明确不想要的内容
- **宽高比**：根据用途选择合适比例

- **首尾帧**：选择连贯的画面
- **创意度**：50-70效果较好

#### 优化图片大小

- 截图工作流分享

## ❓ 常见问题

### Q1: 如何批量导入图片？
**A**: 在左侧故事板区域，可以拖拽多张图片一次性导入。

### Q2: 生成的图片在哪里？
**A**: 生成完成后，图片会显示在节点上，点击节点可以查看大图，在详情面板可以下载。

### Q3: 如何保存工作流？
**A**: 工作流会自动保存到浏览器本地存储，也可以手动导出为JSON文件。

### Q4: 支持哪些图片格式？
**A**: 支持常见的图片格式：JPG、PNG、GIF、WebP等。

### Q5: 视频生成需要多长时间？
**A**: 根据质量和时长不同，通常需要1-5分钟。

### Q6: 如何撤销操作？
**A**: 使用 `Ctrl+Z` (Windows) 或 `Cmd+Z` (Mac) 快捷键。

### Q7: 可以同时运行多个节点吗？
**A**: 可以，但建议不要同时运行太多，避免资源占用过大。

### Q8: 如何删除节点？
**A**: 选中节点后按 `Delete` 或 `Backspace` 键，或使用右键菜单。

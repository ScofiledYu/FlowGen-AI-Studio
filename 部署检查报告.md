```bash

```
✅ 构建产物已生成

### 步骤 3: 启动生产服务器
```bash
# 使用默认端口 8080
npm start

# 或使用自定义端口
PORT=3000 npm start
```

### 步骤 4: 获取服务器 IP 地址
```bash
# Windows
ipconfig

# macOS/Linux
ifconfig
```

### 步骤 5: 配置防火墙
- **Windows**: 控制面板 → 防火墙 → 高级设置 → 入站规则
- **Linux**: `sudo ufw allow 8080/tcp`

### 步骤 6: 局域网访问
在同一局域网内的设备访问：
```
http://[服务器IP]:8080
```
